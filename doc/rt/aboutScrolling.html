About scrolling

A scrollable arrangement

A scroll bar shows when there is a tall element (by sure a DIV) inside another element with a smaller height, with the appropriate overflow style. For example: 

  <style>
    div#rtContainer { height:200px; overflow:auto;}
    div#rtContent   {  }
  </style>
  <div id="rtContainer">
      <div id="rtContent">
        <h1>Content title</h1>
        <p>...</p>
        <p>...</p>
        <p>...</p>
        <p>...</p>
        <p>...</p>
        <p>...</p>
        <p>...</p>
      </div>
  </div>

If the rtContent DIV is taller than the rtContainer DIV then the latter will display a scroll bar.

This arrangement will usually display two vertical scroll bars, one for the BODY element and other for the container.

The BODY should be similarly manipulated in order to make it hide its vertical scroll bar. Its height should be equal or less than the viewport's height, or its overflow CSS property set to hidden.

To check if the content is taller than the container, this function will return a boolean value if the DOM element is overflowed: 

function isOverflowed(element){
    return element.scrollHeight > element.clientHeight 
    || element.scrollWidth > element.clientWidth;
}

It should be adapted to work on overflow-x only.

Also, there is a function that links a click with the text clicked (from SO).
